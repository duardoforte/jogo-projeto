🎮 Como Jogar
Mover para a esquerda:
Pressione a tecla ◀️ Seta Esquerda para andar para a esquerda.

Mover para a direita:
Pressione a tecla ▶️ Seta Direita para andar para a direita.

Pular:
Pressione a barra de espaço para saltar (somente quando estiver em contato com o chão).


▶️ Como Rodar o Jogo
No Linux
Abra um terminal dentro da pasta do projeto e execute:

python3 main.py


No Windows
Abra o Prompt de Comando ou o PowerShell na pasta do projeto e use um destes comandos:

python main.py


DICA: certifique-se de ter o PgZero instalado no seu ambiente Python:

pip install pgzero
